# Acrylic Enclosure for the Va de Retro RAM Tester 2.00a

## Introduction

The [Va de Retro DRAM Tester v2.00](https://www.va-de-retro.com/foros/viewtopic.php?t=8751) by `cacharreo` is a custom DRAM tester based on an Arduino Nano.

This acrylic enclosure for the mentioned DRAM Tester has been designed with the following requirements in mind:
* super simple top/bottom with open sides design
* provide access to required elements during operation
* allow storing the tester in horizontal or vertical position

## Building the enclosure

### Required material

[<img src="images/vdr-dram-tester-2.00a-enclosure-acrylic-build-materials-IMG_7914-annotated.png" width="400"/>](images/vdr-dram-tester-2.00a-enclosure-acrylic-build-materials-IMG_7914-annotated.png)

* One 3mm DINA5 148Ã—210mm acrylic sheet
* 16x M3 6mm flat screws
* 4x M3 nuts
* 4x M3 6mm female-female standoffs
* 4x M3 22mm female-female standoffs
* 4x Button caps 6x6mm (color #1 for left/right buttons)
* 2x Button caps 6x6mm (color #2) for back button
* 2x Button caps 6x6mm (color #3) for OK button

### Assembly instructions

1. Use the provided [CAD design file](enclosure/vdr-dram-tester-2.00-acrylic-enclosure-3mm-rev1a-ready-to-lasercut.dxf) to have the acrylic sheet cut by laser into the required parts.

   Make sure you peel the plastic protections from all the acrylic parts.

[<img src="images/vdr-dram-tester-2.00a-enclosure-acrylic-pieces-IMG_7906-annotated.png" width="512"/>](images/vdr-dram-tester-2.00a-enclosure-acrylic-pieces-IMG_7906-annotated.png)

2. Start by inserting the four M3 6mm screws to the `Bottom` part.

   Make sure the screw heads stay in the correct side of the `Bottom` part by looking at the reference hole marked in green in the picture.
   Add a bit of Blue Tack temporarily to avoid the screws falling off when flipping the `Bottom` part on the next step.

[<img src="images/vdr-dram-tester-2.00a-enclosure-acrylic-build-step-10-IMG_7907-annotated.png" width="512"/>](images/vdr-dram-tester-2.00a-enclosure-acrylic-build-step-10-IMG_7907-annotated.png)

3. Flip the `Bottom` part to make the screw threads available.

[<img src="images/vdr-dram-tester-2.00a-enclosure-acrylic-build-step-20-IMG_7908-annotated.png" width="512"/>](images/vdr-dram-tester-2.00a-enclosure-acrylic-build-step-20-IMG_7908-annotated.png)

4. Screw the four M3 6mm female standoffs to the M3 6mm flat screws on the `Bottom` part.

   Also, remove the temporary Blue Tack used to hold the screws.

[<img src="images/vdr-dram-tester-2.00a-enclosure-acrylic-build-step-30-IMG_7915-annotated.png" width="512"/>](images/vdr-dram-tester-2.00a-enclosure-acrylic-build-step-30-IMG_7915-annotated.png)

5. Add nuts to four M3 6mm flat screws.

   The nuts act as small standoffs to allow the 6mm screws to insert the right distance into the 6mm standoffs.

[<img src="images/vdr-dram-tester-2.00a-enclosure-acrylic-build-step-40-IMG_7917-annotated.png" width="512"/>](images/vdr-dram-tester-2.00a-enclosure-acrylic-build-step-40-IMG_7917-annotated.png)

6. Place and align the assembled DRAM tester board (without the Arduino Nano) on top of the four M3 6mm standoffs, and secure the board with the previously prepared four M3 6mm flat screws with nuts.

[<img src="images/vdr-dram-tester-2.00a-enclosure-acrylic-build-step-50-IMG_7918-annotated.png" width="512"/>](images/vdr-dram-tester-2.00a-enclosure-acrylic-build-step-50-IMG_7918-annotated.png)

7. Add four M3 6mm flat screws to the `Bottom` part using the remaining external holes.

   Again, make sure the screws are correctly inserted, so that the screw threads point upwards.

[<img src="images/vdr-dram-tester-2.00a-enclosure-acrylic-build-step-60-IMG_7919-annotated.png" width="512"/>](images/vdr-dram-tester-2.00a-enclosure-acrylic-build-step-60-IMG_7919-annotated.png)

8. Screw the four M3 22mm standoffs to the four M3 6mm flat screws added to the `Bottom` part on the previous step.

[<img src="images/vdr-dram-tester-2.00a-enclosure-acrylic-build-step-70-IMG_7920-annotated.png" width="512"/>](images/vdr-dram-tester-2.00a-enclosure-acrylic-build-step-70-IMG_7920-annotated.png)

9. Glue together vertically each pair of 6x6mm button caps, and insert a small portion of Blue Tack into each glued button pair to secure the button caps to the buttons.

   We use doubled-height button caps to make them available on the exterior of the enclosure once inserted into the buttons.

[<img src="images/vdr-dram-tester-2.00a-enclosure-acrylic-build-step-74-IMG_7913-annotated.png" height="400"/>](images/vdr-dram-tester-2.00a-enclosure-acrylic-build-step-74-IMG_7913-annotated.png)[<img src="images/vdr-dram-tester-2.00a-enclosure-acrylic-build-step-78-IMG_7912-annotated.png" height="400"/>](images/vdr-dram-tester-2.00a-enclosure-acrylic-build-step-78-IMG_7912-annotated.png)

10. Insert the four double-height button caps prepared on the previous step into the four control buttons of the DRAM tester board.

[<img src="images/vdr-dram-tester-2.00a-enclosure-acrylic-build-step-80-IMG_7921-annotated.png" width="512"/>](images/vdr-dram-tester-2.00a-enclosure-acrylic-build-step-80-IMG_7921-annotated.png)

11. Insert now the programmed Arduino Nano into its socket on the DRAM tester board.

    We need to make sure the DRAM tester board is completely prepared before adding the `Top` part of the enclosure.

[<img src="images/vdr-dram-tester-2.00a-enclosure-acrylic-build-step-90-IMG_7922-annotated.png" width="512"/>](images/vdr-dram-tester-2.00a-enclosure-acrylic-build-step-90-IMG_7922-annotated.png)

12. Add and align the `Top` part on top of the M3 22mm standoffs.

[<img src="images/vdr-dram-tester-2.00a-enclosure-acrylic-build-step-100-IMG_7923-annotated.png" width="512"/>](images/vdr-dram-tester-2.00a-enclosure-acrylic-build-step-100-IMG_7923-annotated.png)

13. Insert the remaining four M3 6mm flat screws into the M3 22mm standoffs to secure the `Top` part to the enclosure.

[<img src="images/vdr-dram-tester-2.00a-enclosure-acrylic-build-step-110-IMG_7924-annotated.png" width="512"/>](images/vdr-dram-tester-2.00a-enclosure-acrylic-build-step-110-IMG_7924-annotated.png)

14. Finally, add the J3 jumper cable to complete the DRAM tester.

[<img src="images/vdr-dram-tester-2.00a-enclosure-acrylic-build-step-120-IMG_7925-annotated.png" width="512"/>](images/vdr-dram-tester-2.00a-enclosure-acrylic-build-step-120-IMG_7925-annotated.png)
